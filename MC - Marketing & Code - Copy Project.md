### Card 1 — Omniaflow

```
Categoria: ERP · Multi-tenant · Fintech
Nome: Omniaflow — Sistema de Gestão Educacional

O desafio:
Uma rede de escolas precisava de um sistema único que gerenciasse múltiplas unidades 
com dados financeiros isolados, integração com gateway de pagamentos e comunicação em tempo real.

O que entregamos:
Desenvolvemos um ERP completo cobrindo secretaria (matrículas, contratos, alunos), 
módulo pedagógico (turmas, presenças, calendário acadêmico) e financeiro 
(mensalidades, PIX, boletos, conciliação automática via webhooks).

Destaques técnicos:
• Multi-tenancy com isolamento total de dados por unidade
• Integração com gateway Asaas: PIX, boletos e conciliação automática
• Tempo real com WebSockets (Laravel Reverb)
• Engenharia financeira com armazenamento em centavos — precisão sem erros de arredondamento
• Controle de acesso granular por role e unidade

Stack: Laravel 12 · Vue 3 · Inertia v2 · TypeScript · Laravel Reverb · Pest 4
```

---

### Card 2 — CIM Base

```
Categoria: SaaS · Corporativo · Gestão
Nome: CIM Base — Plataforma de Gestão Corporativa

O desafio:
Uma empresa precisava de uma plataforma robusta para gestão de leads e operações internas, 
com controle de permissões sofisticado e interface moderna para múltiplos perfis de usuário.

O que entregamos:
Uma SPA corporativa com arquitetura orientada a domínio, sistema RBAC granular 
e mais de 50 componentes de interface acessíveis e reutilizáveis.

Destaques técnicos:
• Arquitetura DDD: Services, DTOs tipados, Contracts e Exceptions — zero lógica nos Controllers
• Type-safety ponta a ponta: rotas tipadas no frontend, validação com Zod, TypeScript estrito
• Sistema de permissões RBAC com Spatie Laravel Permission
• Biblioteca de 50+ componentes UI acessíveis (shadcn-vue + reka-ui)
• Pipeline de qualidade: PHPStan, Rector, Pest, Pint e ESLint

Stack: Laravel 12 · Vue 3 · TypeScript · Tailwind v4 · shadcn-vue · PHPStan
```
